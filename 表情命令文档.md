# 表情命令文档

## 一、命令格式

系统支持三种命令格式：

| 格式 | 示例 | 说明 |
|------|------|------|
| `@命令#` | `@happy#` | 云端指令格式，以 `@` 开头，`#` 结尾 |
| `[stm32]命令` | `[stm32]happy` | ESP32透传格式，以 `[stm32]` 前缀 |
| 纯命令 | `happy` | 直接命令格式 |

---

## 二、表情命令（24个）

表情命令仅控制 OLED 显示，不带动舵机动画。纯表情显示 1~10 秒后自动回到 idle 状态。

### 2.1 表情列表

| 命令名 | 枚举值 | ID | 描述 |
|--------|--------|-----|------|
| `sadness` | `Expression::SADNESS` | 0 | 悲伤：流泪、嘴角下弯 |
| `surprise` | `Expression::SURPRISE` | 1 | 惊讶：瞪大眼、张嘴 |
| `horrified` | `Expression::HORRIFIED` | 2 | 惊恐：瞳孔缩小、眉毛上挑 |
| `furious` | `Expression::FURIOUS` | 3 | 暴怒：眉头紧皱、咬牙切齿 |
| `anger` | `Expression::ANGER` | 4 | 生气：皱眉、眼神锐利 |
| `disgust` | `Expression::DISGUST` | 5 | 厌恶：撇嘴、眯眼 |
| `skeptical` | `Expression::SKEPTICAL` | 6 | 怀疑：斜眼、眉毛不对称 |
| `suspicious` | `Expression::SUSPICIOUS` | 7 | 可疑：眯眼打量、头微倾 |
| `happiness` | `Expression::HAPPINESS` | 8 | 开心：眯眼笑、嘴角上扬 |
| `fear` | `Expression::FEAR` | 9 | 害怕：瞪大眼、眉毛惊恐上挑 |
| `annoyed` | `Expression::ANNOYED` | 10 | 烦躁：半眯眼、不耐烦 |
| `dejected` | `Expression::DEJECTED` | 11 | 沮丧：低头、眼神无光 |
| `pleading` | `Expression::PLEADING` | 12 | 乞求：水汪汪大眼、眉毛下垂 |
| `guilty` | `Expression::GUILTY` | 13 | 愧疚：眼神回避、眉毛内收 |
| `confused` | `Expression::CONFUSED` | 14 | 疑惑：歪头、一侧眉毛上挑 |
| `bored` | `Expression::BORED` | 15 | 无聊：半闭眼、打哈欠 |
| `vulnerable` | `Expression::VULNERABLE` | 16 | 脆弱：眼神闪躲、嘴唇微颤 |
| `disappionted` | `Expression::DISAPPIONTED` | 17 | 失望：眼睑下垂、叹气 |
| `amazed` | `Expression::AMAZED` | 18 | 惊叹：眼睛发亮、嘴巴大张 |
| `tired` | `Expression::TIRED` | 19 | 疲惫：黑眼圈、眼皮沉重 |
| `despair` | `Expression::DESPAIR` | 20 | 绝望：空洞眼神、面无表情 |
| `embarrassed` | `Expression::EMBARRASSED` | 21 | 尴尬：脸红、眼神飘忽 |
| `excited` | `Expression::EXCITED` | 22 | 激动：眼睛发亮、笑容灿烂 |
| `asleep` | `Expression::ASLEEP` | 23 | 睡觉：闭眼、呼吸平稳 |

### 2.2 表情数据结构

```cpp
enum class Expression {
    SADNESS     = 0,
    SURPRISE    = 1,
    HORRIFIED   = 2,
    // ... 共 24 个表情
    ASLEEP      = 23
};

extern const uint8_t oled_expression[][1024];  // 24帧 × 1024字节
```

**显示方式**：128×64 单色位图，SSD1306 page 格式，通过 `showImage()` 显示

## 三、动画命令（7个）

动画命令同时控制 OLED 表情和舵机动作，形成联动效果。

### 3.1 动画列表

| 命令名 | 关联表情 | 舵机动画 | 超时时间 |
|--------|----------|----------|----------|
| `happy` | EXCITED | 头部 80~120° 摆动 + 身体 80~120° 摆动 | 1~5秒 |
| `sad` | SADNESS | 头部归位到 125° + 身体归位到 95° | 1~5秒 |
| `sleep` | ASLEEP | 头部归位到 110° | 永不超时 |
| `idle` | SURPRISE | 头部归位到 100° + 身体归位到 95° | 1~5秒 |
| `no` | FURIOUS | 身体 75~115° 左右摆动，头部固定 100° | 1~5秒 |
| `yes` | HAPPINESS | 头部 85~115° 点头，身体固定 95° | 1~5秒 |
| `caidan` | 动态帧动画 | 无舵机动画（播放喵喵动画） | 永不超时 |

### 3.2 动画详细说明

#### happy（开心）
```
头部摆动：80° → 120° → 80°（循环）
身体摆动：80° → 120° → 80°（循环）
表情：EXCITED（激动）
```

#### sad（悲伤）
```
头部目标：125°（低头）
身体目标：95°（中立）
表情：SADNESS（悲伤）
```

#### sleep（睡觉）
```
头部目标：110°（微微低头）
表情：ASLEEP（闭眼）
特点：永不超时，需手动切换
```

#### idle（空闲）
```
头部目标：100°（中立）
身体目标：95°（中立）
表情：SURPRISE（惊讶）
特点：系统默认状态，所有动画超时后自动返回
```

#### no（否定）
```
身体摆动：75° → 115° → 75°（左右摇头）
头部固定：100°
表情：FURIOUS（暴怒）
```

#### yes（肯定）
```
头部摆动：85° → 115° → 85°（上下点头）
身体固定：95°
表情：HAPPINESS（开心）
```

#### caidan（彩蛋）
```
播放 miaomiao 动画序列（30帧循环）
无舵机动画
表情：动态切换
特点：永不超时，需手动切换
```

## 五、舵机命令

### 5.1 命令格式

```
<舵机名称>-<角度>
```

### 5.2 支持的舵机

| 舵机名称 | 引脚 | TIM通道 | 角度范围 | 默认角度 |
|----------|------|---------|----------|----------|
| `head` | PD13 | TIM4_CH2 | 0~180° | 100° |
| `body` | PD12 | TIM4_CH1 | 0~180° | 95° |

### 5.3 示例

```
head-90      // 头部转到 90°
body-110     // 身体转到 110°
head-180     // 头部转到最大角度
body-0       // 身体转到最小角度
```

### 5.4 PWM 参数

```
频率：50Hz（周期 20ms）
脉宽：1000~2000us
角度计算：pulse = 1000 + (angle / 180) * 1000
```

## 六、命令处理流程

```
用户指令
    ↓
CmdManager::execute(raw)
    ↓
命令格式解析：
    ├─ @xxx# → 提取 xxx
    ├─ [stm32]xxx → 提取 xxx
    └─ 其他 → 原样使用
    ↓
精确匹配（cmd_map_）：
    ├─ LED命令 → LED控制
    ├─ 表情命令 → AnimationManager::show_expression()
    ├─ 动画命令 → AnimationManager::switch_to()
    └─ 无匹配 → 继续
    ↓
舵机命令检测（含 '-'）：
    └─ head-90 → ServoManager::set_angle()
    ↓
返回执行结果（true/false）
```

## 七、超时机制

### 7.1 动画超时

| 动画 | 超时时间 | 超时后行为 |
|------|----------|------------|
| `happy` | 1~5秒（随机） | 切换到 `idle` |
| `sad` | 1~5秒（随机） | 切换到 `idle` |
| `idle` | 1~5秒（随机） | 切换到 `idle`（保持） |
| `no` | 1~5秒（随机） | 切换到 `idle` |
| `yes` | 1~5秒（随机） | 切换到 `idle` |
| `sleep` | 永不超时 | 保持 |
| `caidan` | 永不超时 | 保持 |

### 7.2 纯表情超时

纯表情命令（24个表情）显示 1~10 秒后自动切换到 `idle` 状态。
